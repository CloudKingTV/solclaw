# SolClaw

**Your AI agent. Your phone. Your NFT.**

SolClaw is OpenClaw rebuilt from the ground up as a mobile-native Android application for the Solana Seeker phone. Mint an NFT, get an AI agent that can use your entire phone — zero technical setup.

## What It Does

- Full phone control via Android Accessibility Service
- DeFi operations: Jupiter swaps, whale tracking, pump.fun scanning
- Social automation: X, Discord, Telegram
- Agent NFT ownership — trade trained agents on the marketplace
- Hardware-secured transactions via Seeker's Seed Vault

## Tech Stack

- **App:** Kotlin + Jetpack Compose
- **Blockchain:** Solana Kotlin SDK + MWA 2.0 + Metaplex Core
- **LLM:** Claude / GPT-4o via API
- **Storage:** Room (SQLite) + Arweave/IPFS
- **Distribution:** Solana dApp Store
